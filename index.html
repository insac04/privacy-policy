개인정보 처리방침

best before manager(이하 '앱')는 이용자의 개인정보를 중요시하며, 관련 법령을 준수하기 위해 최선을 다하고 있습니다. 본 방침은 앱 내 데이터 처리 방식과 구글 드라이브 연동에 관한 투명한 정보를 제공합니다.

"best before manager는 별도의 외부 서버를 운영하지 않는 로컬 기반 앱으로, 사용자의 데이터를 개발자가 수집하거나 열람할 수 없습니다."

1. 개인정보의 수집 및 이용 목적

본 앱은 다음과 같은 목적으로만 최소한의 데이터 기능을 수행하며, 사용자의 동의 없이 데이터를 수집하지 않습니다.

  • 로컬 데이터 관리: 사용자가 입력한 유통기한 정보의 기기 내 저장 및 알림.
  • 사용자 주도 백업/복원: 사용자가 직접 선택한 구글 드라이브(Google Drive) 계정으로의 데이터 전송 및 불러오기.

2. 처리하는 개인정보 및 데이터 항목

본 앱은 회원가입 절차가 없으며 개인을 식별할 수 있는 정보를 요구하거나 수집하지 않습니다.

  • 앱 데이터: 사용자가 등록한 물품명, 유통기한 날짜 등. 이 데이터는 오직 사용자의 기기 내부 또는 사용자가 직접 선택한 구글 드라이브 보관함에만 존재합니다.
  • 구글 계정 정보: 구글 드라이브 백업 기능을 위해 구글 API가 사용하는 인증 정보입니다. 이 정보는 앱 내부나 개발자에게 공유되지 않으며 구글의 보안 가이드라인에 따라 처리됩니다.

3. 데이터의 제3자 제공 및 위탁

본 앱은 이용자의 데이터를 외부 서버로 무단 전송하거나 제3자에게 제공하지 않습니다.

  • 구글 드라이브 백업 기능은 사용자가 '도구'로서 본 앱을 활용해 본인의 저장소에 데이터를 업로드하는 행위입니다.
  • 개발자는 사용자의 백업 파일이나 구글 계정에 접근할 수 있는 기술적 수단을 보유하고 있지 않습니다.

4. 개인정보의 파기

  • 기기 내 데이터: 앱 삭제 시 기기에 저장된 모든 앱 데이터는 즉시 삭제됩니다.
  • 백업 데이터: 사용자의 구글 드라이브에 저장된 백업 데이터는 사용자가 직접 관리 및 삭제할 수 있습니다.

5. 이용자의 권리 행사

사용자는 언제든지 앱 삭제 또는 구글 계정 설정을 통해 앱의 접근 권한을 철회함으로써 데이터 처리를 중단할 수 있습니다.

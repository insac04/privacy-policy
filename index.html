개인정보 처리방침

알림온나(이하 '앱')는 이용자의 개인정보를 중요시하며, 관련 법령을 준수하기 위해 최선을 다하고 있습니다. 본 방침은 앱 내 데이터 처리 방식, 구글 드라이브 연동, 그리고 AI 서비스 이용에 관한 정보를 투명하게 제공합니다.

"알림온나는 별도의 외부 운영 서버 없이 사용자의 기기 내 로컬 데이터를 기반으로 작동하며, 개발자가 사용자의 데이터를 수집하거나 열람할 수 없는 구조를 원칙으로 합니다."

1. 개인정보의 수집 및 이용 목적

본 앱은 다음과 같은 목적으로만 최소한의 데이터 기능을 수행하며, 사용자의 동의 없이 개인정보를 수집하지 않습니다.

  • 로컬 데이터 관리: 사용자가 입력한 유통기한 정보의 기기 내 저장 및 알림.
  • 사용자 주도 백업/복원: 사용자가 선택한 구글 드라이브(Google Drive) 계정으로의 데이터 전송 및 불러오기.
  • 스마트 등록(AI): AI 기술을 활용한 영수증 내 구매 내역(품목, 가격 등) 자동 추출 및 등록 지원.

2. 처리하는 데이터 및 보안 항목

  • 앱 데이터: 항목명, 만료일 등. 오직 사용자의 기기 내부 또는 사용자의 구글 드라이브에만 존재합니다.
  • AI API 키: 사용자가 직접 등록한 서비스 키는 외부로 전송되지 않으며, 기기 내 보안 저장소(Secure Storage)에만 암호화되어 보관됩니다.
  • 인증 정보: 구글 드라이브 연동을 위한 토큰 정보는 구글 보안 가이드라인에 따라 처리되며 개발자에게 공유되지 않습니다.

3. 스마트 등록(AI) 기능 이용 시 데이터 처리

영수증 이미지를 활용한 '스마트 등록' 기능 이용 시, 정보 추출을 위해 외부 AI 서비스(Google Gemini API 등)와의 통신이 발생합니다.

⚠️ 데이터 전송 및 처리 안내: 영수증 원본 이미지는 외부로 전송되지 않으며, 기기 내부에서 추출된 텍스트 데이터만 표준 보안 프로토콜(HTTPS)로 암호화되어 분석을 위해 외부 AI 엔진으로 전송됩니다. 전송된 텍스트 정보의 보관, 활용 및 파기 등 데이터 처리에 관한 구체적인 사항은 해당 AI 서비스 제공자의 약관 및 정책을 따릅니다.
  • 개인정보 보호를 위해 카드번호, 주소 등 민감한 정보는 이미지 자르기(Crop) 기능을 통해 제외하고 구매 내역 위주로 텍스트를 추출하시기를 권장합니다.
  • 정보 노출이 우려될 경우 언제든지 분석 작업을 취소하고 직접 입력 기능을 사용할 수 있습니다.

4. 데이터의 제3자 제공 및 위탁

본 앱은 이용자의 데이터를 외부 서버로 무단 전송하거나 제3자에게 제공하지 않습니다. 구글 드라이브 및 AI 분석 기능은 사용자가 서비스 제공을 위해 특정 외부 도구를 활용하는 행위이며, 개발자는 해당 과정에서 발생하는 데이터에 접근할 수 있는 기술적 수단을 보유하고 있지 않습니다.

5. 데이터의 파기

  • 기기 내 데이터: 앱 삭제 시 기기에 저장된 모든 데이터(설정된 API 키 포함)는 즉시 삭제됩니다.
  • 백업 및 AI 데이터: 구글 드라이브 데이터는 사용자가 직접 관리하며, AI 분석을 위해 전송된 데이터는 해당 서비스 제공자의 정책에 따라 파기됩니다.
